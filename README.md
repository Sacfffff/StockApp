# StockApp
Full featured IOS App showcasing Crypto API.

- Write in SwiftUI;
- Using MVVM;
- RestAPI;
- Pagination available;
- Local JSONManager;
- Local Image cache;
- Using Swift Data;
- Using TipKit;
